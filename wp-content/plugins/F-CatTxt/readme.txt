F-Category Text

-----------------------------------------------------------------------------------------------------------------------
Инструкция
-----------------------------------------------------------------------------------------------------------------------

1. скопируйте новый файл F-CatTxt.php в папку /wp-content/plugins/F-CatTxt и замените им старый.
2. скопируйте новый файл aioseop_class.php в папку /wp-content/plugins/all-in-one-seo-pack и замените им старый
3. после замены файла, проверьте админ-панель. Записи->Рубрики->Изменить - в окне редактирования рубрики должны появится новые поля для редактирования:
- Верхнее описание
- Нижнее описание
- SEO Title
- SEO Description
- SEO Keywords
- Шаблон
4. Первые два были в предыдущей версии, добавился стандартный редактор вп с тегами для удобства. В версии 1.1 редактор автоматически добавляет теги p и br
Вывод в шаблоне первого <?php show_descr_top($cat);?>
Вывод в шаблоне второго <?php show_descr_bottom($cat);?>
5. SEO Title - сео заголовок для вашей категории. Будет выводится в теге <title> на странице категории. Будет выводится только в случае если установлен плагин последней версии All in One SEO Pack и выполнен пункт 2.
SEO Description и SEO Keywords - сео описание и ключевые слова для категории. Будут выводится в тегах <meta name="description"> и  <meta name="keywords"> на странице вашей категории.
6. Шаблон. Позволяет задать нужный шаблон для вывода страницы категории. По умолчанию доступен “стандартный вывод” - шаблон, который используется по умолчанию в вашей теме для вывода страниц категории (index.php, category.php и другие). 
Чтобы создать новый шаблон достаточно создать файл с названием category-название_шаблона.php в директории вашей темы. В самом начале файле должен быть следующий текст:
<?php
/**
 * Category Template: Название_вашего_шаблона
 */
?>
Пример файла шаблона категории category-fseo.php - можно скопировать в папку с вашей темой. Его содержание:
<?php
/**
 * Category Template: FSeoCat
 */
?>
<?php get_header(); // выводим стандартный хедер ?>
<?php show_descr_top($cat); // выводим верхнее описание категории ?>
<?php show_descr_bottom($cat); // выводим нижнее описание категории ?>
<?php get_footer(); // выводим стандартный футер ?>
